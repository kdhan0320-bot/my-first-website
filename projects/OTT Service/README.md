# STREAMIX (OTT Service) — 프론트엔드 UI 데모

## 개요
OTT 서비스 화면을 참고해 만든 정적 프론트엔드 UI 데모입니다. Hero, 콘텐츠 카드,
장르 필터, 예고편 모달, 게스트 체험 안내 모달 등 OTT형 서비스에서 흔히 쓰이는
화면 구성과 인터랙션을 구현하는 데 목적이 있습니다.

## 기술 스택
- HTML
- CSS (`css/style.css`)
- Vanilla JavaScript (`js/main.js`), 별도 프레임워크나 빌드 도구 없음

## 구현된 인터랙션
- 헤더 스크롤에 따른 배경 전환
- 모바일 햄버거 메뉴 열기/닫기
- 장르 필터 버튼으로 콘텐츠 카드 필터링(결과 없음 상태 포함)
- 예고편 모달 열기/닫기, 포커스 트랩, ESC 닫기
- 찜하기 토글(Hero, 콘텐츠 카드), 내 리스트 추가 토글(상세 섹션)
- 스크롤 위치에 따른 nav 링크 active 전환
- Hero의 "상세 정보" 버튼 클릭 시 상세 섹션으로 스크롤
- 카드 재생 버튼 또는 Enter / Space 입력으로 정적 예고편 UI 모달 열기(클릭한 카드 제목이 모달에 반영됨)
- 게스트 체험 안내 모달 열기/닫기, 포커스 트랩, ESC 닫기

## 정적·게스트 데모 방식
모든 콘텐츠(제목, 장르, 평점, 에피소드 수 등)는 HTML에 하드코딩된 정적 데이터입니다.
"게스트로 체험하기" 버튼은 실제 로그인 상태를 만들지 않고 안내 모달을 닫아
메인 화면을 바로 보여주는 방식으로만 동작합니다.

## 외부 API
사용하지 않습니다. 서버, 데이터베이스, 외부 API 호출이 전혀 없는 순수 정적 페이지입니다.

## 구현하지 않은 기능
- 실제 회원가입 및 로그인 기능
- 실제 결제 및 스트리밍(영상 재생) 기능

## 한계
- 찜하기, 내 리스트 추가 등 상태 변경 내용은 저장되지 않으며, **새로고침하면 초기화**됩니다.
- 상태는 `localStorage`/`sessionStorage` 없이 메모리(DOM 클래스)로만 관리됩니다.

## 실행 방법
별도 설치나 빌드 과정이 없습니다. `index.html`을 브라우저로 열어서 확인하면 됩니다.
