# Mini SNS

교육 과정에서 제작한 초기 결과물을 바탕으로, 게스트 데모 흐름과 mock 데이터, 프론트엔드 상태 관리를 개인적으로 재정리한 프로젝트입니다.

- 배포 주소: https://kdhan0320-bot.github.io/dohan-portfolio/mini-sns/

---

## 제작 목적

모바일 SNS 앱의 핵심 화면 흐름을 학습하기 위해 교육 과정에서 처음 만든 결과물을, 실제 백엔드 없이도 주요 인터랙션을 체험할 수 있도록 게스트 데모 흐름과 mock 데이터, 프론트엔드 상태 관리(React Context) 구조로 개인적으로 재정리했습니다.

홈 피드, 게시글 작성, 프로필, 모임, 채팅, 알림 화면을 하단 탭 내비게이션으로 연결해 사용자가 여러 기능을 자연스럽게 이동할 수 있도록 구성했습니다.

---

## 주요 기능

- 게스트 데모 진입
- 홈 피드
- 게시글 작성 / 수정 / 삭제 (메모리 상태)
- 피드 이미지 확대 모달
- 좋아요 토글 (메모리 상태)
- 댓글 작성 / 수정 / 삭제 (메모리 상태)
- 프로필 화면
- 모임 목록
- 모임 참가 후 채팅방 이동
- 채팅 목록 / 채팅방 상세
- 메시지 입력창 하단 고정
- 알림 화면
- 하단 탭 내비게이션
- 반응형 모바일 앱 레이아웃

## 게스트 데모

로그인 화면의 `게스트 데모 시작하기` 버튼을 누르면 별도의 회원가입·로그인 절차 없이 바로 주요 화면을 체험할 수 있습니다.

- 표시 이름: 데모유저
- 아이디 표기: @demo_user

---

## 데모 데이터와 한계

이 프로젝트는 취업용 포트폴리오 데모이며 실제 운영 서비스가 아닙니다.

- 실제 회원가입·로그인 기능이 없습니다.
- 자체 백엔드와 데이터베이스를 사용하지 않습니다.
- 게시글·댓글·좋아요 변경은 브라우저 메모리에서만 동작하며, 프로필은 준비된 데모 정보를 표시합니다.
- 새로고침하면 처음 준비된 mock 데이터로 복원됩니다.
- 모임·채팅·알림 화면도 모두 mock 데이터로 구성되어 있으며, 실시간 통신 기능은 없습니다.
- 실제 이미지 업로드, 실시간 채팅, 푸시 알림 기능은 제공하지 않습니다.

## 참고 사항

이 프로젝트는 모바일 SNS 앱의 핵심 화면 흐름을 보여주기 위한 학습용 데모 프로젝트입니다. 실제 운영 서비스가 아니므로 게스트 데모와 mock 데이터는 시연 목적에 맞게 구성되어 있습니다.
